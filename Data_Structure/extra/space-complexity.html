공간 복잡도

공간 복잡도란?

공간 복잡도(Space Complexity)는 알고리즘이 실행되는 동안 추가로 사용하는 메모리 공간의 크기를 입력 크기 n에 따라 표현한 것이다.

공간 복잡도에서 고려하는 요소

  • 입력 데이터를 제외한 추가 메모리 사용량
  • 변수, 배열, 객체 등 자료구조
  • 재귀 호출 시 사용되는 호출 스택

대표적인 공간 복잡도

  • O(1) : 입력 크기와 무관한 고정된 공간
  • O(n) : 입력 크기에 비례하는 공간
  • O(n²) : 2차원 배열 등 중첩 구조

예시

크기 n의 배열을 하나 더 생성하는 알고리즘은 O(n)의 공간 복잡도를 가진다.

반면, 몇 개의 변수만 사용하는 알고리즘은 O(1) 공간 복잡도를 가진다.

공간 복잡도가 중요한 이유

메모리는 한정된 자원이기 때문에, 대용량 데이터나 임베디드 환경에서는 시간 복잡도만큼이나 공간 복잡도도 중요한 평가 기준이 된다.
