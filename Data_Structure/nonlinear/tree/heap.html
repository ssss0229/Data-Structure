힙의 정의

힙은 완전 이진 트리 기반의 자료구조로, 최댓값이나 최솟값을 빠르게 찾기 위해 사용됩니다.

  • 우선순위 큐를 구현하는 데 가장 효율적입니다.
  • 느슨한 정렬 상태(반 정렬 상태)를 유지합니다.

힙의 종류

최대 힙 (Max Heap)

부모 노드의 키값이 자식 노드보다 크거나 같은 힙

최소 힙 (Min Heap)

부모 노드의 키값이 자식 노드보다 작거나 같은 힙

힙의 배열 표현

힙은 완전 이진 트리이므로 배열로 효율적으로 구현할 수 있습니다.

  • 부모 인덱스: $i // 2$
  • 왼쪽 자식 인덱스: $i * 2$
  • 오른쪽 자식 인덱스: $(i * 2) + 1$

힙의 연산

삽입과 삭제 모두 O(log n)의 시간 복잡도를 가집니다.

  • 삽입 (Up-heap): 마지막에 노드를 추가하고 부모와 비교하며 위로 올립니다.
  • 삭제 (Down-heap): 루트 노드를 제거하고 마지막 노드를 루트로 올린 뒤 자식과 비교하며 내립니다.

Python heapq 라이브러리


import heapq

heap = []

# 원소 삽입 (최소 힙 기준)
heapq.heappush(heap, 10)
heapq.heappush(heap, 5)
heapq.heappush(heap, 20)

# 최솟값 추출
print(heapq.heappop(heap)) # 5
